# Careersphere AI

Welcome to **Careersphere AI**! This application was developed to revolutionize the career and recruitment process using cutting-edge technologies.

## Features
- AI-driven Mock Interviews
- Recruiter Workspace
- Candidate Profiles
- High-fidelity premium design aesthetics

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set your environment variables in `.env` (like `GEMINI_API_KEY` and Firebase credentials).
3. Run the app:
   `npm run dev`
